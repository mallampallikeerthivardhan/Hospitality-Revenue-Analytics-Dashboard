# Hospitality Revenue Analytics Dashboard

## Overview
Developed an interactive Power BI dashboard to analyze hotel performance, occupancy trends, booking behavior, and revenue metrics.

## Tools Used
- Power BI
- SQL
- Power Query
- Excel

## Key KPIs
- Revenue
- Occupancy %
- RevPAR
- ADR
- Realization Rate

## Key Insights
- Identified top-performing properties across multiple cities.
- Analyzed booking and cancellation trends.
- Evaluated revenue performance and occupancy metrics.
- Generated business insights to support decision-making.
